# Quiz 16 - Tendências e Futuro do MCP ✨

1. Por que o MCP é chamado de "O Futuro da Interoperabilidade em IA"?
    - [ ] Porque ele foi criado por viajantes do tempo.
    - [x] Porque ele remove as barreiras de integração, permitindo que qualquer modelo de cada empresa fale com qualquer dado.

    - [ ] Porque ele é o protocolo oficial da Lua.
    *Explicação: O fim das APIs proprietárias em favor de um padrão aberto é a grande tendência.*

2. O que é o "Marketplace de Servidores MCP"?
    - [ ] Uma feira livre para vender computadores.
    - [x] Um ecossistema (repositório) onde desenvolvedores compartilham e instalam capacidades pré-prontas para suas IAs.

    - [ ] O nome do escritório da Anthropic.
    *Explicação: Compartilhar conectores (GitHub, PostgreSQL, Slack) acelera a adoção da IA.*

3. Como a "Soberania de Dados" se beneficia do MCP?
    - [ ] Permitindo que os dados sejam vendidos para o exterior.
    - [x] Permitindo que empresas usem IA de ponta sem nunca enviar seus dados privados para os servidores de treinamento das IAs.
    - [ ] Criando novas leis de trânsito digital.
    - [ ] Impedindo o uso de IAs internacionais.
    *Explicação: O MCP permite manter os dados locais e enviar apenas o contexto necessário para a IA responder.*

4. O que são "Autonomous Agents" e qual o papel do MCP neles?
    - [ ] Robôs que limpam a casa; o MCP é o controle remoto.
    - [x] IAs que decidem e executam tarefas longas; o MCP fornece as "ferramentas" para que elas ajam no mundo real.
    - [ ] Funcionários humanos que não precisam de chefes.
    - [ ] Carros que dirigem sozinhos (apenas).
    *Explicação: O MCP é o conjunto de capacidades que torna a autonomia da IA possível.*

5. Qual a previsão para o desenvolvimento de software nos próximos anos com o MCP?
    - [ ] Os programadores vão parar de escrever código.
    - [x] O foco mudará de "construir integrações" para "arquitetar contextos e ferramentas para IAs".
    - [ ] O código será escrito apenas em papel.
    - [ ] Não haverá mais internet.
    *Explicação: O papel do desenvolvedor evolui para facilitador da inteligência.*

6. Como o MCP ajuda na "Democratização da IA"?
    - [ ] Dando um computador para cada habitante.
    - [x] Permitindo que pequenos desenvolvedores criem ferramentas que funcionam instantaneamente com os maiores modelos do mundo.
    - [ ] Diminuindo o preço da energia elétrica.
    - [ ] Traduzindo todos os livros para todas as línguas.
    *Explicação: Padrões abertos nivelam o campo de jogo para todos.*

7. O que é o conceito de "Edge MCP"?
    - [ ] Usar o navegador Microsoft Edge para o MCP.
    - [x] Rodar servidores MCP diretamente em dispositivos móveis ou locais, sem depender da nuvem.
    - [ ] Usar o MCP no limite da velocidade da luz.
    - [ ] Um comando para deletar a beirada (edge) do monitor.
    *Explicação: Processamento local (Edge) aumenta a privacidade e reduz a latência.*

8. Qual o maior desafio ético do avanço dos Agentes com MCP?
    - [ ] A IA ficar muito triste.
    - [x] O risco de delegar decisões críticas (financeiras/legais) para a IA sem supervisão adequada.
    - [ ] A IA aprender a jogar videogame.
    - [ ] O protocolo MCP ser muito difícil de aprender.
    *Explicação: Ética e responsabilidade devem crescer junto com a autonomia técnica.*

9. Por que o conhecimento de MCP é um diferencial para os gestores de tecnologia (CTOs)?
    - [ ] Para eles poderem consertar os computadores dos funcionários.
    - [x] Pois permite estruturar estratégias de "AI-First" que são seguras, escaláveis e não dependem de um único fornecedor.

    - [ ] Para eles poderem programar em Python.
    *Explicação: Visão estratégica sobre padrões garante a longevidade tecnológica da empresa.*

10. Qual a mensagem principal do encerramento deste curso sobre o futuro?
    - [ ] O futuro é sombrio e difícil.
