# Projeto 16 - Servidor MCP "Future-Proof" ✨

## Objetivo
Explorar as fronteiras do protocolo MCP, implementando suporte a múltiplos transportes e extensibilidade.

## Requisitos
- Projeto integrador anterior.

## Atividades
1. **Híbrido Stdio/SSE**: Modifique seu servidor para que ele possa ser iniciado tanto via terminal quanto via HTTP, dependendo de uma variável de ambiente.
2. **Marketplace Ready**: Documente seu servidor seguindo os padrões para publicação em repositórios comunitários de servidores MCP.
3. **Reflexão Ética**: Escreva um breve parágrafo no `README` sobre as diretrizes de segurança e privacidade dos dados processados pelo seu servidor.

## Entrega
- Código atualizado com suporte a múltiplos transportes.
- Documento de "Manifesto de Servidor" (Objetivos, Segurança e Uso).
